# Poynting Effect Simulator

포인팅 효과 시뮬레이터는 고무와 같은 탄성 원기둥을 비틀 때 높이와 반지름이 어떻게 변하는지 보여주는 웹 기반 시뮬레이션입니다. Zurlo et al. (2020)의 이론식을 바탕으로, 비틀림만 있는 경우, 질량이 매달린 상태에서 비틀림이 함께 작용하는 경우, 토크를 직접 제어하는 경우를 비교할 수 있습니다.

핵심 목적은 "비틀림이 단순히 회전 변형만 만드는 것이 아니라, 축방향 길이 변화도 만들 수 있다"는 점을 시각적으로 확인하는 것입니다.

## 01. 입력 변수

시뮬레이션에 사용할 기본 물리량을 정하는 부분입니다.

- 초기 높이 `H`: 변형 전 원기둥의 높이입니다. 높이가 클수록 같은 비틀림에서 상대적인 늘어남이 작아집니다.
- 초기 반지름 `R_e`: 변형 전 원기둥의 반지름입니다. 반지름이 클수록 비틀림에 따른 축방향 효과가 커집니다.
- 전단 탄성률 `mu`: 재료가 전단 변형에 얼마나 저항하는지를 나타냅니다. 값이 클수록 단단한 재료입니다.
- 매달린 질량 `m`: 원기둥 아래에 작용하는 하중을 정합니다. 질량이 있으면 비틀지 않아도 평형 높이가 달라질 수 있습니다.
- 가한 토크 `M`: 토크 제어 모드에서 원기둥에 직접 가하는 모멘트입니다. 이 모드에서는 비틀림 각도 `theta`가 입력이 아니라 결과로 계산됩니다.
- 최대 비틀림 `theta / 2pi`: 애니메이션과 그래프에서 사용할 최대 회전 수입니다. 1 turn은 360도 회전을 뜻합니다.

## 02. 계산 모드

같은 입력값을 세 가지 물리 상황으로 계산할 수 있습니다.

- 무하중: 매달린 질량이 없는 순수 비틀림 상황을 보여줍니다. 이 모드에서는 비틀림에 의한 포인팅 효과를 중심으로 볼 수 있습니다.
- 하중 + 비틀림: 매달린 질량에 의한 축방향 인장과 비틀림 효과를 함께 계산합니다. 기본 설정은 이 모드입니다.
- 토크 제어: 비틀림 각도를 직접 주지 않고 토크 `M`을 입력해 평형 높이와 결과 비틀림 각도를 계산합니다.

`변형 전 기준 원기둥` 옵션은 변형 전 원기둥을 회색으로 함께 표시합니다. 현재 상태가 원래 형태와 얼마나 다른지 비교하기 위한 기준입니다.

## 03. 애니메이션

현재 비틀림 값을 0에서 최대 비틀림까지 자동으로 변화시키는 부분입니다.

- 재생: 비틀림을 점점 증가시키며 원기둥의 형상 변화를 보여줍니다.
- 정지: 현재 비틀림 상태에서 애니메이션을 멈춥니다.
- 초기화: 입력값과 비틀림 상태를 기본값으로 되돌립니다.

애니메이션은 설명을 돕기 위한 기능이며, 계산 자체는 애니메이션을 실행하지 않아도 입력값이 바뀔 때마다 즉시 다시 수행됩니다.

## 04. 저장

시뮬레이션 결과를 외부에서 다시 확인하거나 보고서에 넣기 위한 기능입니다.

- CSV 데이터: 입력 조건, 현재 상태, 그래프에 사용된 계산값을 표 형태로 저장합니다.
- PNG 그림: 현재 3D 시각화 화면을 이미지로 저장합니다.

## 05. 3D 시각화

비틀린 원기둥의 현재 형상을 보여주는 중심 화면입니다.

주황색 원기둥은 현재 계산된 평형 상태입니다. 회색 원기둥은 옵션을 켰을 때 표시되는 변형 전 기준 형상입니다. 화면 아래의 `현재 비틀림` 슬라이더를 움직이면 원하는 회전 수에서 높이, 반지름, 늘어난 길이를 바로 확인할 수 있습니다. 토크 제어 모드에서는 이 슬라이더가 입력이 아니라 계산된 결과 비틀림을 표시합니다.

이 섹션에서 말하려는 핵심은 다음과 같습니다.

- 비틀림이 증가하면 원기둥이 단순히 꼬이는 것뿐 아니라 높이도 변할 수 있습니다.
- 질량이 매달려 있으면 전단 탄성률과 하중의 영향 때문에 비틀지 않은 상태에서도 평형 높이가 달라집니다.
- `총 신장 h-H`와 `비틀림 추가 신장 h(theta)-h(0)`를 분리해 보여줍니다. 그래서 자연 길이 기준의 전체 늘어남과, 하중을 먼저 건 뒤 비틀림 때문에 추가로 생긴 늘어남을 따로 확인할 수 있습니다.
- 초기 부피와 현재 부피를 함께 보여주어, 비압축성 재료 가정에서 반지름과 높이가 서로 연결되어 변한다는 점을 확인할 수 있습니다.

## 06. 그래프

비틀림의 크기에 따라 물리량이 어떻게 변하는지 곡선으로 보여주는 부분입니다.

그래프는 특정 순간만 보는 3D 화면과 달리, 0부터 최대 비틀림까지의 전체 경향을 보여줍니다. 무하중 결과와 하중이 있는 결과를 비교하면, 매달린 질량이 포인팅 효과의 관찰값에 어떤 차이를 만드는지 더 쉽게 볼 수 있습니다.

특히 확인할 점은 다음과 같습니다.

- 비틀림이 커질수록 높이 변화가 어떻게 누적되는지
- 반지름이 높이 변화와 함께 어떻게 줄어드는지
- 무하중과 하중 조건에서 곡선의 차이가 어떻게 나타나는지

## 07. 지배 방정식

시뮬레이션 계산에 사용된 핵심 이론식을 정리한 부분입니다.

- 무하중 비틀림 식은 질량이 없을 때 비틀림만으로 생기는 높이 변화를 설명합니다.
- 가늘고 긴 근사는 반지름이 높이에 비해 충분히 작을 때 사용할 수 있는 단순화된 식입니다.
- 하중 + 비틀림 식은 매달린 질량에 의한 힘 `F`와 전단 탄성률 `mu`를 함께 포함한 평형 조건입니다.
- 토크 제어 식은 모멘트 `M`을 입력으로 두고 높이와 결과 비틀림 `theta`를 계산합니다.

이 방정식 섹션은 시각화가 단순한 그림이 아니라, 어떤 물리 모델을 바탕으로 계산되는지 보여주기 위한 근거입니다.

## 사용 방법

1. `index.html`을 브라우저에서 열거나 GitHub Pages 주소로 접속합니다.
2. 입력 변수에서 원기둥의 높이, 반지름, 전단 탄성률, 질량, 최대 비틀림을 조절합니다.
3. 계산 모드를 선택합니다.
4. 3D 시각화와 그래프에서 높이 변화, 반지름 변화, 부피비를 확인합니다.
5. 필요한 경우 CSV 또는 PNG로 결과를 저장합니다.

```bash
git add .

git push
```

push 후 GitHub Pages가 다시 배포되기까지 보통 1분 정도 걸릴 수 있습니다.
